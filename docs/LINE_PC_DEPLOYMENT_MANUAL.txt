===============================================================================
  CONNECTOR VISION SOP AGENT v4.5.0
  Line PC Deployment Manual
===============================================================================

This manual reflects the active deployment path only.

ACTIVE PACKAGE CONTENTS
-----------------------
- connector_vision_agent.exe
- start_agent.bat
- stop_ollama.bat
- assets\config.json
- assets\sop_steps.json
- assets\models\yolo26x.pt
- assets\models\yolo26x_local_pretrained.pt

IMPORTANT
---------
- The app bundle is GUI-first.
- `start_agent.bat` must open the PyQt6 GUI.
- Archived pretrain datasets and pretrain bundles are not required for normal
  line deployment.

DEPLOYMENT
----------
1. Copy the app bundle to the line PC.
2. Keep the folder structure intact.
3. Double-click `start_agent.bat`.
4. Confirm the GUI opens.
5. Use `Tab 1 - Run SOP` for production verification.

FINE-TUNING
-----------
1. Use `Tab 7 - Training`.
2. Start from `yolo26x_local_pretrained.pt`.
3. Reload the new model after training.
4. Verify the result in `Tab 1 - Run SOP`.

SOP EDITING
-----------
1. Use `Tab 4 - SOP Editor`.
2. Review changes before saving.
3. Validate edited steps with a controlled SOP run.

ARCHIVED PRETRAIN PATH
----------------------
If a historical rebuild of the pretrained seed is ever needed, see:
- `legacy/pretrain/README.md`
- `scripts/run_pretrain_local.py`

Do not use that path for standard field deployment.
