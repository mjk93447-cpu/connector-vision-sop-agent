================================================================
Connector Vision SOP Agent v7.0.0 — Deployment Merge Guide
================================================================

Merge order for a complete offline line PC pack:

  1. Extract connector-agent-app-cpu.zip OR connector-agent-app-gpu.zip
  2. Extract connector-agent-llm-local-cache.zip into the same root
     (creates ollama_models\ or llm_stage\ with blobs + manifests)
  3. Run install_first_time.bat as Administrator once
  4. Launch start_agent.bat

Required models in the LLM cache:

  qwen3:8b    — SOP Generate (4-pass atomizer)
  gemma4:9b   — LLM Chat + SOP recovery

Config reference: assets\config.json

  llm.model_path              = gemma4:9b
  llm.sop_generation.model_path = qwen3:8b

If only one model is present, the GUI still opens but SOP Generate or
LLM Chat will show a readiness warning until the missing tag is staged.
