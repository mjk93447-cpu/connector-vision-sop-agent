================================================================
 Connector Vision SOP Agent v3.0 - All-in-One Package
 INSTALL GUIDE
================================================================

STEP 1.  phi4-mini 모델 블롭 복사 (약 2.7 GB)
--------------------------------------------------
기존 빌드에서 portable-part2-phi4-mini 아티팩트를 다운로드합니다.
  Actions run:
  https://github.com/mjk93447-cpu/connector-vision-sop-agent/actions/runs/23139568715

압축 해제 후 blobs\ 와 manifests\ 폴더를 아래 경로에 복사:

  [이 패키지 폴더]\ollama_models\blobs\
  [이 패키지 폴더]\ollama_models\manifests\

STEP 2.  실행
--------------------------------------------------
  start_agent.bat 더블클릭

  => Ollama 서버 자동 시작 (포트 11434)
  => phi4-mini-reasoning 모델 로드 확인
  => PyQt6 GUI 오픈 (6탭)

최종 폴더 구조
--------------------------------------------------
  connector_agent\
  +-- connector_vision_agent.exe   <- SOP Agent (GUI + YOLO26x)
  +-- ollama.exe                   <- LLM 런타임
  +-- start_agent.bat              <- 실행 스크립트 (더블클릭)
  +-- stop_ollama.bat
  +-- INSTALL_GUIDE.txt            <- 이 파일
  +-- assets\
  |   +-- config.json              <- 설정 (편집 가능)
  |   +-- sop_steps.json           <- SOP 12단계 정의 (편집 가능)
  |   +-- models\
  |       +-- yolo26x.pt           <- YOLO 가중치 (CI=플레이스홀더)
  +-- ollama_models\
      +-- blobs\     <- Part2에서 복사
      +-- manifests\ <- Part2에서 복사

GUI 6탭 구성
--------------------------------------------------
  Tab1: Run SOP    - 12단계 SOP 자동 실행 / 로그 실시간 확인
  Tab2: Vision     - 스크린샷 캡처 + YOLO bbox 오버레이 / 파일 열기
  Tab3: LLM Chat   - LLM에게 SOP 실패 원인 질문 / config 변경 제안
  Tab4: SOP Editor - SOP 단계 추가/삭제/순서 변경 (JSON 직접 편집)
  Tab5: Config     - config.json 값 UI로 편집
  Tab6: Audit      - config 변경 이력 열람

NOTES
--------------------------------------------------
  - LLM 없이도 Tab1/Tab2/Tab5 정상 동작
  - yolo26x.pt: CI 플레이스홀더 (실제 학습 모델로 교체 시 검출 성능 향상)
  - sop_steps.json 편집 후 EXE 재시작 없이 즉시 반영
  - config.json -> llm.allow_config_write=true 설정 시
    LLM이 직접 config.json 수정 가능 (감사 로그 기록됨)
