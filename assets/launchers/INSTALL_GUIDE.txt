================================================================
Connector Vision SOP Agent v7.0.0 - Full App Packs
 INSTALL GUIDE
================================================================

PACKAGE ARTIFACTS
--------------------------------------------------
  1. connector-agent-app-cpu.zip
  2. connector-agent-app-gpu.zip
  3. connector-agent-llm-local-cache.zip (optional offline LLM bundle)

REQUIRED OFFLINE LLM MODELS (v7.0.0)
--------------------------------------------------
  SOP Generate : qwen3:8b   (llm.sop_generation in config.json)
  Chat/Recovery: gemma4:9b  (llm.model_path in config.json)

  Lite RAM fallback: qwen3:4b for SOP Generate only.

FUNCTION SHARING POLICY
--------------------------------------------------
  CPU and GPU app packs share the same application code:
  - PyQt6 GUI (SOP Generate, SOP Run, SOP Editor, Training, LLM Chat)
  - 4-pass document atomizer (PDF/PPTX/TXT/MD -> runtime SOP steps)
  - YOLO26x vision + OCR-first control

INSTALL
--------------------------------------------------
  1. Create deployment folder, e.g. C:\tools\connector_agent\
  2. Extract one app pack (cpu or gpu).
  3. Extract LLM bundle into the same root as ollama_models\ or llm_stage\.
  4. Run install_first_time.bat once as Administrator (registers models).
  5. Double-click start_agent.bat.

FINAL FOLDER STRUCTURE
--------------------------------------------------
  connector_agent\
  +-- connector_vision_agent.exe
  +-- start_agent.bat
  +-- install_first_time.bat
  +-- ollama.exe
  +-- ollama_models\   (blobs\ + manifests\)
  +-- assets\
  |   +-- config.json          (version 7.0.0)
  |   +-- sop_steps.json
  |   +-- models\
  |       +-- yolo26x_local_pretrained.pt
  +-- README_INSTALL_EN.md
  +-- docs\V7_0_0_FOCUS.md

RUNTIME BEHAVIOR
--------------------------------------------------
  - start_agent.bat restores split LLM chunks automatically.
  - SOP Generate tab requires qwen3:8b (or configured sop_generation tag).
  - LLM Chat and recovery_action use gemma4:9b.
  - See docs/V7_0_0_FOCUS.md for operator workflow.
