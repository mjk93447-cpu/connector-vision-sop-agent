================================================================
 Connector Vision SOP Agent v3.2.5 - All-in-One Package
 INSTALL GUIDE
================================================================

STEP 1.  Copy phi4-mini model blobs (~2.7 GB)
--------------------------------------------------
Download the connector-agent-llm artifact from the latest build:
  GitHub Actions:
  https://github.com/mjk93447-cpu/connector-vision-sop-agent/actions

Extract the ZIP and copy the blobs\ and manifests\ folders to:

  [this package folder]\ollama_models\blobs\
  [this package folder]\ollama_models\manifests\

STEP 2.  Run first-time setup (once only)
--------------------------------------------------
  Right-click install_first_time.bat -> Run as administrator
  Wait for "Setup Complete!" message (2-5 minutes)

STEP 3.  Launch
--------------------------------------------------
  Double-click start_agent.bat

  => Ollama server starts automatically (port 11434)
  => phi4-mini-reasoning model loads
  => PyQt6 GUI opens (7 tabs)

Final folder structure
--------------------------------------------------
  connector_agent\
  +-- connector_vision_agent.exe   <- SOP Agent (GUI + YOLO26x)
  +-- ollama.exe                   <- LLM runtime
  +-- start_agent.bat              <- Launch script (double-click)
  +-- stop_ollama.bat              <- Stop Ollama server
  +-- install_first_time.bat       <- First-time setup (run once as Admin)
  +-- INSTALL_GUIDE.txt            <- This file
  +-- QUICK_START_EN.md            <- One-page daily reference
  +-- README_INSTALL_EN.md         <- Full user guide
  +-- assets\
  |   +-- config.json              <- Settings (edit via Tab 5)
  |   +-- sop_steps.json           <- SOP 12-step definition (edit via Tab 4)
  |   +-- models\
  |       +-- yolo26x.pt           <- YOLO26x weights (COCO pretrained baseline)
  +-- ollama_models\
      +-- blobs\     <- copy from connector-agent-llm artifact
      +-- manifests\ <- copy from connector-agent-llm artifact

GUI 7-Tab Layout
--------------------------------------------------
  Tab 1: SOP Runner  - Run 12-step SOP / real-time log
  Tab 2: Vision      - Screenshot + YOLO bbox overlay / file open
  Tab 3: LLM Chat    - Ask LLM about SOP failures / config suggestions
  Tab 4: SOP Editor  - Add/delete/reorder SOP steps (JSON)
  Tab 5: Config      - Edit config.json via UI (saves to config.proposed.json)
  Tab 6: Audit       - Config change history viewer
  Tab 7: Training    - Annotate connector images and fine-tune YOLO26x

NOTES
--------------------------------------------------
  - Tabs 1/2/5 work without LLM (Ollama not required for core SOP)
  - yolo26x.pt: COCO pretrained by default; fine-tune via Tab 7 for better accuracy
  - sop_steps.json edits take effect immediately (no restart needed)
  - config.json changes: always use Tab 5 -> review config.proposed.json -> apply manually
  - Training: second run "NoneType write" error is fixed in v3.2.5 (stale cache auto-cleared)
  - LLM first response may take 60-120 seconds on CPU-only hardware (normal)

REQUIREMENTS
--------------------------------------------------
  - Windows 10 Pro 1803+ or Windows 11 (64-bit)
  - 1920x1080 display resolution (REQUIRED)
  - 16 GB RAM minimum
  - 15 GB free disk space
