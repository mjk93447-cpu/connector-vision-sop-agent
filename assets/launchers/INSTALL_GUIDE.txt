================================================================
Connector Vision SOP Agent v5.0.0 - App + CUDA Overlay
 INSTALL GUIDE
================================================================

PACKAGE ARTIFACTS
--------------------------------------------------
  1. connector-agent-app-core.zip
     - connector_vision_agent.exe
     - launchers, config, SOP steps, docs
     - lightweight runtime pieces that do not include torch CUDA payloads
  2. connector-agent-cuda-runtime.zip
     - torch / torchvision / nvidia runtime folders
     - CUDA-related DLLs required by the packaged app and local training flows

OPTIONAL MANUAL ARTIFACTS
--------------------------------------------------
  - connector-agent-pretrain.zip
  - connector-agent-llm.zip

CANONICAL ENTRYPOINTS
--------------------------------------------------
  - Main app: src/gui_app.py -> connector_vision_agent.exe
  - Training tab fine-tuning promotes the latest runs\detect\train\weights\best.pt
    into assets\models\yolo26x_local_pretrained.pt automatically
  - Local pretrain prefers CUDA first on NVIDIA PCs and now fails loudly when
    a CUDA wheel is missing instead of silently dropping to CPU
  - Model naming reference: docs/MODEL_ARTIFACT_NAMING.md

ASSEMBLY
--------------------------------------------------
  1. Extract connector-agent-app-core.zip into the final deployment folder.
  2. Extract connector-agent-cuda-runtime.zip into the same folder.
  3. Keep the extracted folder structure unchanged.
  4. Optional: merge connector-agent-pretrain.zip and connector-agent-llm.zip
     only when those offline features are needed on the target PC.

RUN
--------------------------------------------------
  - Double-click start_agent.bat to launch the main SOP agent.
  - If ollama.exe is absent, the launcher continues in non-LLM mode.

FINAL FOLDER STRUCTURE
--------------------------------------------------
  connector_agent\
  +-- connector_vision_agent.exe
  +-- _internal\
  |   +-- torch\
  |   +-- torchvision\
  |   +-- nvidia\
  +-- start_agent.bat
  +-- stop_ollama.bat
  +-- assets\
  |   +-- config.json
  |   +-- sop_steps.json
  |   +-- models\
  |       +-- yolo26x_local_pretrained.pt
  +-- PLACE_APP_HERE.txt
  +-- PLACE_CUDA_RUNTIME_HERE.txt   <- only remains if CUDA overlay was not merged
  +-- README.md
  +-- README_INSTALL_EN.md

NOTES
--------------------------------------------------
  - The packaged runtime model slot is assets\models\yolo26x_local_pretrained.pt.
  - The core artifact alone is not a runnable GPU package until the CUDA overlay
    is merged into the same folder.
